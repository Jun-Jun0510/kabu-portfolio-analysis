# 全世界株式（オルカン）の役割と最適配分分析

## エグゼクティブサマリー

**結論: eMAXIS Slim 全世界株式はポートフォリオに残すべき。S&P500は全世界株式に統合（売却→全世界に振替）を推奨。**

推奨配分（インデックス部分）:
- eMAXIS Slim 全世界株式: **100%**（現在の311,620円 + S&P500からの343,000円 = 約655,000円）
- eMAXIS Slim S&P500: **0%**（売却して全世界に統合）

理由: FANG+で米テック集中投資を既に持っている以上、インデックス部分はS&P500ではなく全世界で「非米国分散」を最大化すべき。

### 重複度の判定

- 全世界株式の約64%は米国株 = S&P500と大きく重複
- **両方持つと、ポートフォリオ全体の米国比率がさらに上昇**
- 現在の保有比率（全世界:S&P500 = 48:52）だと、実質米国比率は約83%

**判定: 両方持つ意味は薄い。テック集中（FANG+）を既に持つなら、インデックス部分は非米国分散を最大化すべき。**

## 2. 非米国株が提供する分散効果（定量評価）

### 2025年のパフォーマンス実績

| 資産クラス | 2025年リターン（USD建て） |
|-----------|--------------------------|
| 非米国先進国株（MSCI EAFE） | +31.2% |
| 新興国株 | +30%前後 |
| オルカン（全世界） | -4.0%（円建てYTD 4月時点） |
| S&P500 | +17.9%（USD）/ -7.8%（円建てYTD 4月時点） |

### 重要な発見

1. **2025年は非米国株が米国株を10%以上アウトパフォーム**
2. 新興国株は9ヶ月間でオルカンを9%、S&P500を13%上回った
3. 欧州株はドイツ・EUの財政拡張策で上昇基調
4. 円建てでも全世界 > S&P500（2025年YTD: -4.0% vs -7.8%）

### 相関係数の低下

- 非米国株と米国株の相関: 0.74（3年間、以前は0.93）
- **相関が低下 = 分散効果が増大中**

## 3. マクロ環境の分析（ドル安トレンド）

### 現在の為替環境
- USD/JPY: 158.5円（円安だが、ドル指数は下落中）
- DXY: 98.3（ドル安トレンド）
- 2025年上半期: ドルは1991年以来最大の半年下落（約-11%）

### ドル安が意味すること

1. **非米国資産のドル建てリターンが押し上げられる**
2. 新興国通貨が相対的に強くなり、新興国株にポジティブ
3. 欧州ユーロ上昇 → 欧州企業の競争力に一部マイナスだが、資産価値は上昇
4. 「米ドル離れ」の思惑が新興国市場への資金流入を促進

### 結論
ドル安環境では、全世界株式の非米国部分（約36%）がパフォーマンスの改善ドライバーとなる。S&P500のみでは、このドル安メリットを享受できない。

## 4. FANG+ + 全世界 = バーベル戦略としての評価

### バーベル戦略の構造

```
[集中・高リスク]          [分散・安定]
FANG+ (米テック10銘柄)  ←→  全世界株式 (約3,000銘柄)
高ボラティリティ             低~中ボラティリティ
セクター集中                 セクター・地域分散
```

### なぜバーベル戦略として優秀か

1. **FANG+が取るリスク**: 米テック10銘柄への等金額集中投資。AI/クラウド/広告の成長に賭ける
2. **全世界が取るリスク**: 広範な地域・セクター分散。世界経済全体の成長に連動
3. **相互補完性**: FANG+が不調な時（テック規制、バリュエーション調整）、全世界の非テック部分がクッションに

### S&P500ではバーベルにならない理由

- S&P500のM7比率は31%超 → FANG+と大きく重複
- S&P500 + FANG+ = 「テック超集中 + テック集中」であり、バーベルではなく**片寄りダンベル**
- 全世界は非米国36%を含むため、FANG+との真のバーベルが成立

### 有効性の評価: ★★★★☆（5段階中4）

バーベル戦略として有効。唯一の懸念は、全世界の米国比率64%がまだ高いこと。しかし、純粋な非米国ファンドよりも管理が容易で、長期保有に適している。

## 5. 全世界とS&P500の両方を持つべきか？

### 結論: **S&P500は不要。全世界に一本化すべき。**

### 理由

| 観点 | 両方持つ場合 | 全世界のみ |
|------|-------------|-----------|
| 米国比率 | 83%（過剰） | 64%（適正） |
| FANG+との重複 | 高い | 低い |
| 非米国分散 | 17%のみ | 36% |
| 管理の手間 | 2本 | 1本 |
| ドル安メリット | 限定的 | 享受可能 |
| コスト | 同等 | 同等 |

### 「S&P500を残すべき」唯一の反論とその反駁

- 反論: 「米国の長期優位性に賭けたい」
- 反駁: FANG+を既に保有している時点で、米テック・米国株への過剰エクスポージャーは十分すぎる。さらにS&P500を加えるのは「米国一極集中」のリスクを不必要に高める

## 6. 推奨配分比率

### インデックスファンド部分の最適配分

```
【現在】
全世界: 311,620円 (48%)  +  S&P500: 343,000円 (52%) = 654,620円

【推奨】
全世界: 654,620円 (100%) +  S&P500: 0円 (0%)
```

### ポートフォリオ全体における全世界の位置づけ

全世界株式は「安定成長のコア」として、ポートフォリオ全体の30-40%を担うべき。

具体的な役割分担:
- **FANG+/QQQ/個別テック**: 攻めの成長（高リスク・高リターン）
- **全世界株式**: 守りのコア（中リスク・中リターン、地域分散）
- **債券/キャッシュ**: 防御（低リスク）

### 積立の方向性

新規積立（NISAつみたて投資枠）は全世界株式に集中させる。理由:
1. 長期（20年+）ではテック集中よりも分散が安全
2. NISAの非課税メリットを最大化するには、長期保有に適した分散型が最適
3. FANG+は成長投資枠で別途対応
