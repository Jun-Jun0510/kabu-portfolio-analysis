# メタトレンド評価：中嶋聡の視点による個別銘柄分析

## 分析フレームワーク

「AIがソフトを飲み込む」時代のメタトレンドとは何か。

1. **AIファースト・アーキテクチャ**: すべてのソフトウェア、サービスがAIをコアに再設計される
2. **フィジカルAI（物理AI）**: AIがデジタル世界を超えて物理世界を制御する（自動運転、ロボット、エネルギー）
3. **エッジコンピューティングの台頭**: 推論がクラウドからデバイスへ移行する
4. **エネルギーがAIの最終ボトルネック**: 計算資源の爆発的増大が電力需要を根本的に変える

## Tier 1: メタトレンドの勝者（プラットフォーム・Moat保有）

### GOOGL（Alphabet）: 買い増し
**設計思想の美しさ: A** / **Moat: S** / **プラットフォーム性: S**

Alphabetは「AIがソフトを飲み込む」メタトレンドの最大の体現者。Gemini 3.1 Pro、Google Cloud 48%成長、Waymo週50万回有料ライド到達。これは単なる事業の寄せ集めではなく、**検索→クラウド→自動運転→量子コンピューティング**という一貫した「知能のインフラ」を構築している設計思想の美しさがある。

- Geminiアプリ7.5億MAU：これは**AIのOS化**への道筋
- Waymo 10都市展開：Tesla FSDよりデータの質で優位（完全無人走行の実績）
- Google Cloud 48%成長：AIインフラ需要が供給を超過

**Moatの本質**: データフライホイール。検索データ→AI学習→より良い検索→より多くのデータ。このサイクルは指数関数的に強化される。

### AMZN（Amazon）: 買い増し
**設計思想の美しさ: A** / **Moat: S** / **プラットフォーム性: S**

Amazonの2026年CapEx $2000億は狂気に見えるが、これは**AIインフラのデファクトスタンダード**を取りに行く賭け。BedrockがAIのOSになりつつある：10万社が利用、Q1 2026のトークン処理量がそれ以前の全期間合計を超過。

- Trainium2チップ140万枚稼働：NVIDIA依存からの脱却
- 100万台のロボット稼働：物流の物理AIプラットフォーム化
- Bedrock = AIのAWS化：モデル選択のプラットフォーム支配

**Moatの本質**: 3層の参入障壁（EC物流網＋クラウドインフラ＋AIプラットフォーム）。各層が相互に強化し合う。

### TSLA（Tesla）: 保持（条件付き買い増し）
**設計思想の美しさ: B+** / **Moat: A** / **プラットフォーム性: A**

Teslaは**自動車メーカーからフィジカルAIプラットフォーム企業への変態**の途上。FSD v14がMotorTrend年間最優秀賞を獲得、ロボタクシーがダラス・ヒューストンに拡大、Optimus第3世代が2026年後半に量産開始。

- FSD累計84億マイル（Waymoの42倍の走行データ量）
- ロボタクシーがオースティンで安全監視員なしで走行開始
- Optimus：マスクは将来の企業価値の80%を占めると発言

**設計思想の懸念**: マスクの政治的関与によるブランド毀損リスク。ただし、FSDの物理世界データの蓄積は構造的に不可逆。ユーザーがモータ制御の専門家であることを考慮すると、Teslaのインバータ技術（SiCパワーデバイス統合設計）の先進性は技術的に理解しやすいはず。

**条件**: 政治リスクが許容できれば買い増し。そうでなければ保持。

## Tier 2: メタトレンドに乗る有望銘柄

### AMD: 買い増し
**設計思想の美しさ: A** / **Moat: B+** / **プラットフォーム性: B+**

MI350/MI450シリーズはNVIDIAへの構造的な対抗軸。データセンターGPU収益が前年比114%成長で$150億予測。HBM4搭載MI400シリーズとHeliosラックスケールプラットフォームは設計として美しい。

- MI450: 432GB HBM4メモリ、MI350から50%増
- Heliosプラットフォーム: 72基MI455Xで31TB HBM4
- Microsoft、Meta、OpenAIが採用コミット

**Moatの限界**: NVIDIAのCUDAエコシステムという巨大なソフトウェアMoatに対して、ハードウェア性能だけでは不十分。ただし「NVIDIA一極集中」を嫌う顧客ニーズが構造的に存在する。

### ARM: 保持
**設計思想の美しさ: S** / **Moat: A** / **プラットフォーム性: A**

35年間のIP専業からの大転換：初の自社製造シリコン「ARM AGI CPU」を発表。Metaが最初の顧客。ハイパースケーラーの計算能力の50%がARMベース。

- 設計思想は最も美しい（最小限の命令セットで最大効率）
- エッジAI時代のロイヤリティ上昇（チップ価値の2.5%→5%）
- ロイヤリティ収入過去最高6.2億ドル（前年比21%増）

**リスク**: RISC-Vが世界市場の25%を獲得。長期的にはオープンソースアーキテクチャとの競争が最大の脅威。現在の株価はかなり織り込み済み。

### D（Dominion Energy）: 保持
**設計思想の美しさ: B** / **Moat: A** / **プラットフォーム性: B**

AIデータセンターのエネルギー需要という構造的テーマの直接的受益者。契約容量48.5GW、2026-2030年で$647億の設備投資計画。

- Alphabet、Amazon、Microsoft、Metaと直接契約
- AmazonとのSMR共同開発（300MW、5億ドル）
- 洋上風力CVOW 70%完成

**メタトレンドとの関係**: 「エネルギーがAIの最終ボトルネック」テーマの安定した恩恵銘柄。ただしユーティリティの設計思想に革新性はない。安定配当＋成長のバランス型。

### QCOM（Qualcomm）: 買い増し
**設計思想の美しさ: B+** / **Moat: A** / **プラットフォーム性: B+**

エッジAIのメタトレンドにおける最有力企業の一つ。Snapdragon X2 Elite（85 TOPS）、自動車収益$11億（前年比15%増）、デザインウィン・パイプライン$450億。株価は年初来25%下落で割安。

- CEO「エッジAIの勝者がAI全体の勝者になる」
- 自動車＋IoT収益目標：2029年までに$220億
- $200億の自社株買い発表

**エンジニア視点**: ユーザーの専門であるモータ制御・インバータの世界でも、エッジAI推論によるリアルタイム制御最適化は重要なトレンド。QCOMのSnapdragonプラットフォームはその計算基盤になり得る。

## Tier 3: 要精査・リスク銘柄

### OKLO: 保持（リスク許容なら）
**設計思想の美しさ: A** / **Moat: B** / **プラットフォーム性: C**

設計思想は美しい：小型モジュラー原子炉でAIデータセンターの電力問題を解決。MetaとSwitch（12GW）との大型契約、2026年7月4日に臨界達成目標。

- DOEが安全設計合意書を承認
- 液性資産$26億で資金面は安泰
- アナリスト14名平均「Strong Buy」、目標$99.58

**しかし**：まだ1ワットも発電していない。臨界達成は「目標」であり、原子力の規制リスクは予測不能。設計は美しいがMoatはまだ構築途上。ハイリスク・ハイリターン。ポートフォリオ内のポジションサイズ（約10万円）は適切。

### MRAM（Everspin Technologies）: 保持
**設計思想の美しさ: B+** / **Moat: B+** / **プラットフォーム性: C**

不揮発性メモリ（MRAM）はニッチだが構造的に重要な技術。238デザインウィン（前年178から増加）、UNISYST MRAM新製品投入、自動車グレード対応。

- Q4 2025売上$1480万、通年$5520万
- 新製品が車載・産業向けに展開中
- +57.92%のリターンは出来過ぎ

**メタトレンドとの関係**: エッジAI・自動車のリアルタイム処理には不揮発性高速メモリが不可欠。ただし市場規模が小さく、大手半導体が参入すれば競争激化。ニッチMoat銘柄として少額保持は合理的。

### RXRX（Recursion Pharmaceuticals）: 保持（忍耐銘柄）
**設計思想の美しさ: A** / **Moat: B+** / **プラットフォーム性: B**

AI創薬プラットフォームとして設計思想は美しい。REC-4881のFAP Phase 1b/2で初の臨床的バリデーション達成。Sanofi、Roche、Bayer、Merkとの提携で$5億以上。

- 5つの臨床プログラム、約15の探索段階プログラム
- REC-394 Phase 2データ、REC-1245 Phase 1データが2026年前半に
- 現金ランウェイは2027年まで

**メタトレンドとの関係**: 「AIがバイオを飲み込む」メタトレンドの先駆者。ただし創薬は成功確率が構造的に低い。-24.29%の含み損だが、臨床データが出るまでは忍耐。

### DNA（Ginkgo Bioworks）: 売り
**設計思想の美しさ: B** / **Moat: C** / **プラットフォーム性: B-**

設計思想は当初美しかった（合成生物学のAWS）が、実行が伴っていない。バイオセキュリティ事業売却、Q4売上$3300万（前年比24%減）、年間損失$3.13億。OpenAIとの自律ラボ実験はアピール的だが本業の縮小は深刻。

- 売上減少が続いている（通年25%減）
- バイオセキュリティ事業を売却（主力事業の一角を失う）
- 現金$4.23億だが赤字が大きく消耗速度が速い

**構造的問題**: 合成生物学プラットフォームのMoatが証明されていない。顧客が自社でラボを構築する「内製化リスク」が高い。これは「美しい設計思想が市場に受け入れられなかった」ケースに近い。損切り推奨。

### ONON（On Holding）: 売り
**設計思想の美しさ: B+** / **Moat: B** / **プラットフォーム性: C**

プロダクトとしては美しい（CloudTecソール技術）。ただし**メタトレンドとの関連性が極めて低い**。AIもエッジコンピューティングもエネルギーも関係ない。

- FY26ガイダンス23%成長だがウォール街予想を下回る
- CEO交代（共同創業者がCo-CEOに）
- ピークから30%下落

**メタトレンド視点**: テクノロジスト・ポートフォリオにおいて、この銘柄を保有する構造的理由がない。成長は見込めるが、AIメタトレンドのポートフォリオには不適合。売却してメタトレンド銘柄に再配分すべき。

## 投資信託の評価

### iFreeNEXT FANG+: 保持
FANG+はメタトレンドの中核銘柄を含むが、集中度が高い。S&P500やオルカンとの重複を考慮すると、これ以上の追加は不要。

### eMAXIS Slim S&P500 / 全世界株式: 保持
コア資産として適切。ドルコスト平均法での継続投資が望ましい。

### 楽天・全米株式: 保持
+89.41%のリターン。米国全体への分散投資として優秀。

### ゴールド・ファンド（為替ヘッジあり）: 保持～買い増し
金価格$4,769まで上昇、GS目標$5,400。スタグフレーション懸念時のヘッジとして有効。為替ヘッジありは円高リスク対策として合理的だが、ポートフォリオ全体でのゴールド比率は他チームメンバーの分析に委ねる。

## エンジニアへの補足

ユーザーの専門（モータ制御・インバータ・ベクトル制御）との接点：

1. **TSLA**: SiCインバータの統合設計、FSD用の物理AIフレームワークはモータ制御のリアルタイム最適化と通底
2. **QCOM**: Snapdragonのエッジ推論（85 TOPS）は産業用モータ制御AIの計算基盤候補
3. **AMD**: Xilinx（Versal ACAP）によるFPGA＋AI推論はモータ制御のデジタルツイン実現に寄与
4. **MRAM**: モータ制御ECUの不揮発性高速メモリとして技術的に重要
