## 3. 特に答えるべき5つの問い

### Q1: NVDAとAVGOのどちらが「今買い」としてバリュエーション的に妥当か？

**結論: NVDA**

| 比較項目 | NVDA | AVGO |
|----------|------|------|
| 予想P/E | 24.18x | 31〜37x |
| PEGレシオ | 0.64 | 0.25〜0.73 |
| 営業利益率 | 60.38% | 未公開（推定35-40%） |
| 売上成長率 | +65% | AI部門+150%（全社はそこまでではない） |
| リスク | 中国輸出規制 | VMware統合リスク |

NVDAは予想P/E 24xで65%成長、PEG 0.64。AVGOのAI部門成長は凄いが、全社ベースの予想P/Eが31-37xとより高い。現時点では**NVDAの方がバリュエーション的に妥当**。

ただし、分散効果を狙うなら両方を持つのが最適。NVDAを先行投入し、AVGOは決算後の調整で追加が理想。

### Q2: OKLO（売上ゼロ）を10.6万円分保有する合理性は数字的にあるか？

**結論: 合理性は低い**

- 時価総額 $8.7-11.3B に対して売上ゼロ
- 収益開始は2030年以降（4年以上先）
- $2.6Bの手元資金はあるが、年$350-450M消費
- ポートフォリオの約17%をゼロ収入企業に配分は過大
- **推奨**: 半分（5万円程度）に縮小し、残りをNVDA等に再配分
