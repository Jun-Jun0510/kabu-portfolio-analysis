## 2. テック70%集中リスクの評価

### 定量評価

| 指標 | 数値 | 評価 |
|------|------|------|
| テック/AI関連比率 | 70% | 高集中 |
| 単一セクター最大ドローダウン（過去例） | -35%（2022年NASDAQ） | 最悪ケースで約140万円の評価減 |
| 回復期間（2022→2024 ATH） | 約2年 | 長期投資なら許容範囲 |
| FANG+年率リターン（2019-2025） | +28%/年 | リスクに見合うリターン実績 |

### 許容可否の判定

**結論: 条件付きで許容可能**

許容する根拠:
1. **年齢・時間軸**: 育休世代 → 10年以上の投資期間がある。-35%ドローダウンからの回復に2年かかっても問題ない
2. **資産規模**: 280万円は生活防衛資金とは別（前提）。全損しても生活に直接影響なし
3. **テーゼの一貫性**: 「AIのクラウド→エッジ→物理世界浸透」という明確な投資仮説がある
4. **FANG+の内部分散**: 10銘柄均等加重で、NVIDIA一極集中ではない

許容するための条件:
1. ゴールド10% + 現金6% = 16%の非テック資産を確保
2. S&P500 14%でヘルスケア・金融・エネルギーへの間接エクスポージャー
3. 「-30%ルール」: ポートフォリオ全体が-30%を超えたら自動的にリスク削減を検討
4. 追加投資余力（給与等）があれば下落時に買い増し
